    • Bluestar!

    Scientific name is Am. tabernaemontana, likes full to partial sun. Grows around three feet.

    Butterfly's favor this plant!
    Butterfly Weed!

    Scientific name is Asclepias Tuberosa, likes full sun. Will grow 1-2 feet.

    Butterfly's favor this plant!
    Aster!

    Scientific name is Aster Tataricus. Likes partial sun, will grow 1-5 feet.

    Will grow in low nitrogen soil and it attracts butterflies!
    Tickseed!

    Scientific name is Coreopsis. Likes full to partial sun, there are multiple varieties with different ranges of growth.

    A+ from pollinators!
    Purple Coneflower!

    Scientific name is Echinacea purpurea. Likes full sun, will grow to be 3-5 feet.

    Our pollinatiors like these!
    Joe-Pye-Weed

    Scientific name is Eupatorium spp. Likes full sun, will grow to be 5-9 feet.

    Attracts butterflies!
    Purple-headed Sneezeweed!

    Scientific name is Helenium flexuosum. Likes full sun, will grow to be 1-4 feet.

    Attracts bees and butterflies!
    Swamp Sunflower!

    Scientific name is Helianth. angustifolius. Likes full sun, will grow to be 5-7 feet. Good for pollinators!

    Blazing Star

    Scientific name is Liatris spp. Likes full sun and will grow to be 2-5 feet.

    Hummingbirds and other pollinators like this one!
    Wild Bergamot!

    Scientific name is Monarda fistulosa. Likes full to partial sun, will grow 2-5 feet.

    Pollinators enjoy this one, but so do wasps!
    White beard-tongue!

    Scientific name is Penstemon digitalis. Likes full sun, will grow to be 3-4 feet.

    Bees like this one!
    Summer Phlox!

    Scientific name is Phlox paniculata. Likes full to partial sun, will grow 2-4 feet.

    Pollinator friendly!
    Silver Leaf Mountain Mint!

    Scientific name is Pycnanthemum incanum. Likes full sun, will grow to be 2-4 feet.

    This plant can be used medicinally!
    Black-eyed Susan!

    Scientific name is Rudbeckia fulgida. Likes full sun, will grow to be 2-3 feet.

    Honeybees and butterflies like this plant!
    Wrinkle-leaf Goldenrod!

    Scientific name is Solidago rugosa. Likes full sun, will grow to be 3-7 feet.

    Highly sought after by pollinators!
    Ironweed!

    Scientific name is Vernonia gigantea. Likes full sun, will grow to be roughly 7 feet.

    Moderately attractive to bees
